# Output Schema

This document defines the CSV outputs generated by `python main.py`.

## Data Source Behavior
- Default mode (`--buildings-source osm`): buildings are fetched live from OpenStreetMap using `data/manual/campus_boundary.geojson`.
- Optional CSV mode (`--buildings-source csv`): buildings are loaded from `--buildings-csv`.

## `data/processed/buildings.csv`
| Column | Type | Description |
|---|---|---|
| `building_name` | string | Source building name. |
| `latitude` | float | Building latitude in WGS84. |
| `longitude` | float | Building longitude in WGS84. |
| `building_id` | string | Stable slug generated from `building_name`. |
| `source` | string | Building source (`osm` or `csv`). |

## `data/processed/graph_edges.csv`
| Column | Type | Description |
|---|---|---|
| `edge_id` | string | Stable edge identifier `from-to-key`. |
| `from_node` | int/string | Source graph node ID. |
| `to_node` | int/string | Destination graph node ID. |
| `distance_m` | float | Edge distance in meters. |
| `travel_mode` | string | Current mode, default `walk`. |
| `surface` | string/null | OSM surface tag when available. |
| `one_way` | bool | Whether edge is one-way. |

## `data/processed/routes.csv`
| Column | Type | Description |
|---|---|---|
| `origin_building_id` | string | Origin building slug. |
| `destination_building_id` | string | Destination building slug. |
| `algorithm` | string | `dijkstra` or `astar`. |
| `distance_m` | float | Shortest path distance in meters. |
| `estimated_time_min` | float | Estimated walk time at configured speed. |
| `path_node_count` | int | Number of nodes in shortest path. |
| `path_nodes` | string | `|`-joined ordered node IDs. |
| `path_buildings` | string | `|`-joined origin and destination names. |
| `computed_at` | ISO datetime | UTC timestamp for route computation. |

## `data/processed/validation_summary.md`
Human-readable summary with generation timestamp and row counts for:
- buildings
- graph edges
- routes
